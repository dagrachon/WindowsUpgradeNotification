This Powershell Script creates a Windows 10 Notification-Toast (Notification for your Action-Center).

It's used for reminding the user to install a Windows Feature Upgrade, more Details at https://github.com/dagrachon/windows_update_script.

You may use this script as base configuration for your own scripts to use Windows Notification Toasts.
